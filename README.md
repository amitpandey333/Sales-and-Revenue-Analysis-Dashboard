# 📊 Sales & Revenue Analysis Dashboard

An interactive **Sales & Revenue Analysis Dashboard** built using **Microsoft Power BI** to analyze sales performance, revenue trends, and business insights.

---

## 📌 Project Overview

This dashboard helps businesses monitor sales performance through interactive visualizations and key performance indicators (KPIs). It allows users to explore sales data using filters and slicers to make better business decisions.

---

## 🎯 Project Objectives

- Analyze overall sales and revenue performance.
- Track monthly sales trends.
- Identify top-performing products.
- Compare sales across different regions and categories.
- Provide interactive filtering for better analysis.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Microsoft Excel
- GitHub

---

## 📂 Dataset

**Source:** Sample Superstore Dataset (Excel)

The dataset contains information such as:
- Order ID
- Order Date
- Product Name
- Category
- Region
- Sales
- Profit
- Quantity

## 📈 Dashboard Features

### KPI Cards
- 💰 Total Sales
- 💵 Total Profit
- 📦 Total Orders

### Charts
- 📈 Monthly Sales Trend (Line Chart)
- 📊 Top Products by Sales (Bar Chart)
- 🥧 Sales by Category (Pie Chart)
- 📍 Sales by Region (Column Chart)

### Interactive Filters
- Region
- Category
- Segment

4. Use the slicers to interact with the dashboard.
